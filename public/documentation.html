myFlix API Documentation

This API provides users access to information about different movies, directors, and genres. Users can sign up, update their personal information, and create a list of their favorite movies on myFlix.

Business Logic Endpoint URL HTTP Method Query Parameters Request body data format Response body data format
Get a list of all movies /movies GET none none A JSON object holding a list of all movies
Get data about a single movie by title /movies/:title GET title none A JSON object holding data (title, description, genre, director, image URL, featured or not) about a specific movie title
Get data about a genre by name/title (e.g., "Thriller") /movies/genre/:genreName GET genreName none A JSON object holding a description of the genre of a specific movie
Get data about a director by name /movies/directors/:directorName GET directorName none A JSON object holding data (name, bio, birth year, death year) of a specific director
Add a user /users POST none A JSON object holding data about the user to add, structured like: { "name": "Jordan", "favoriteMovies": [] } A JSON object holding data about the user that was added
Update a user's info (username) /users/:id PUT id A JSON object holding data about the user's updated info, structured like: { "name": "Jordan Updated Name", "favoriteMovies": [] } A JSON object holding data about the updated user
Add a movie to user's list of favorites (showing only a text that a movie has been added) /users/:id/:movieTitle POST id, movieTitle none A text message indicating whether the movie was successfully added
Remove a movie from user's list of favorites (showing only a text that a movie has been added) /users/:id/:movieTitle DELETE id, movieTitle none A text message indicating whether the movie was successfully removed
Remove a user (showing only a text that a user email has been removed) /users/:id DELETE id none A text message indicating whether the user was successfully removed
